สวัสดีพี่ๆทุกคนด้วยครับ ผมชุติพงศ์ เอี้ยงปราณี (ปั๊ม) ผมกำลังศึกษาอยู่มหาวิทยาลัยศรีปทุม คณะเทคโนโลยีสารสนเทศและการสื่อสาร ผมไม่เก่งพวกฐานข้อมูลเท่าไหร่ พวกเขียนเว็ปไซต์ผมก็ไม่กล้าบอกด้วยว่าผมเก่ง ผมเรียน python flask java html มาถึงจุดนี้ผมไม่สามารถบอกได้ว่าตัวเองเก่ง ผมไม่สามารถพิมพ์โค้ดทั้งหมดได้ตั้งแต่ 1 จนถึง 100 แต่ผมเข้าใจหลักการทำงานของมัน สุดท้ายที่ผมจะสื่อผมอยากพัฒนาตัวเองให้เก่งขึ้นครับ.
